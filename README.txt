== Description ==

Product extra-options with dynamic pricing for WooCommcerce : 
Custom Product Manager turns any shop product into a customizable product. 

Allows to define product extra-options and relative costs.
Options will be displayed on the product page, cart page, checkout page, review-order & email order. 
The price will fluctuate dynamically depending on the options set after user interaction.

**Free version** : 2 compilable fields with 14 types of options available.
**Pro version** : 7 compilable fields with 14 types of options available.

1.   TEXT
2.   SELECT DROPDOWN => from 2 to 6 select options.
3.   CHECKBOXES => from 1 to 4 checkboxes.
4.   TEXTAREA 
5.   RADIO BUTTONS => from 2 to 4 radio buttons.
6.   NUMBER 
7.   IMAGES => from 2 to 4 Images.
8.   EMAIL 
9.   BUTTONS => from 3 to 6 buttons.
10.  COLOR 
11.  CALCULATION with custom formula => from 1 to 2 calcul inputs.
12.  URL 
13.  AJAX FILE upload => 100 file mime types available.
14.  PHONE

== Requirements ==

*    Wordpress installed (online or local web server).
*    WooCommerce installed and activated (you can skip store settings).
*    Product regular price must be fixed before to start.

== Installation ==

1. Unzip 'custom-product-manager' and paste its contents into the folder `/wp-content/plugins/`
2. Installed and activate the plugin through the Wordpress plugins menu.

== Usage ==

1. Choose or create a product
2. Select "custom product manager" in the product data panel. 
3. For each field, choose the type of options and the corresponding information.
4. Click on "Save and display option *" and go on the product page or refresh it. 
5. The settings tab allows you to define the type of cost calculation and some text strings on the product page and cart page.
